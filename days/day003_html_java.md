- 1. html/css/js 역할

  - 2) css  :  외관 , 꾸미기
  - 3) js   :  움직임 , 동작

- 2. 태그
  - <> </> : 브라우저에게 명령을 내리는것
  - div    : 영역을 나눠줌
  - h      : 제목 / 1 ~ 6 까지
  - p      : 문단
  - pre    : 줄바꿈을 포함하는 문단
  - img    : 이미지 , src="경로" , alt="설명"
  - a      : 링크 , href="경로" , title="설명"
