EX4: 수동 이미지 슬라이더

        1. 문제: 수동 이미지 슬라이더 만들기
        다음 조건을 만족하는 HTML + JavaScript 코드를 작성하시오.
        - 화면 중앙에 이미지 슬라이더 박스를 만든다.
        - 박스 안에는 하나의 이미지를 표시하며, 크기는 600px × 350px으로 한다.
        - 이미지는 배열에 저장된 여러 개의 URL 중 하나를 보여준다.
        - 박스 아래에 두 개의 버튼을 만든다.
        ◀ PREV 버튼: 이전 이미지를 보여준다.
        NEXT ▶ 버튼: 다음 이미지를 보여준다.

        - 버튼 클릭 시 기본 동작(링크 이동)은 막아야 한다. (e.preventDefault() 사용)

        2. 동작 조건:
        - images 배열에 4개의 이미지 URL을 저장한다.
        - index 변수를 이용해 현재 이미지를 추적한다.
        - PREV 버튼 클릭 시 index를 감소시키고, NEXT 버튼 클릭 시 index를 증가시킨다.
        - 배열의 끝에 도달하면 다시 처음으로 돌아가도록 순환 처리한다.

        3. 출력 예시 화면:
        - 초기 화면: 첫 번째 이미지가 중앙에 표시된다.
        - PREV 버튼 클릭 시 이전 이미지가 표시된다.
        - NEXT 버튼 클릭 시 다음 이미지가 표시된다.
        - 끝까지 이동하면 다시 처음 이미지로 돌아간다.
